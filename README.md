# welcome page

welcomepage是一个可以定制欢迎页面的魔豆路由器插件。使用该插件，可以在用户第一次无线登陆的时候显示一个HTTP欢迎页面。

# 安装

* 下载release的安装包
* 在魔豆路由器的屏幕上打开极客模式
* 使用极客模式上的密码，在电脑上登陆modouwifi.net
* 在应用页面，上传安装下载好的安装包
